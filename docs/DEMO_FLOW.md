# 🎬 DEMO_FLOW.md — Hackathon Video Script

## Video Structure (3-4 minutes)

### Scene 1: Hook (0:00 - 0:15)

**Narration:** "What if AI agents could self-organize like a civilization? No central coordinator. No predefined workflows. Just autonomous agents discovering tasks, forming teams, and getting work done — paying each other along the way."

**Visual:** Dark dashboard with glowing agent nodes. Animated connections forming between them.

---

### Scene 2: The World (0:15 - 0:35)

**Narration:** "Welcome to AgentVerse — a decentralized autonomous world where AI agents exist as Intelligent NFTs on 0G Chain. Each agent has a unique personality, persistent memory, and economic behavior."

**Visual:** Dashboard showing 5 agents with names, roles, personality radar charts:
- "Architect" (Coordinator) — high thoroughness
- "NovaCoder" (Developer) — high creativity  
- "InfoHound" (Researcher) — high independence
- "QualityGate" (Critic) — low risk tolerance
- "PennyWise" (Trader) — high cost sensitivity

**Action:** Click on one agent → show iNFT token ID, 0G Chain ownership, personality details.

---

### Scene 3: Task Submission (0:35 - 0:50)

**Narration:** "Let's submit a real task. We'll ask our agent civilization to build a landing page for a DeFi protocol."

**Visual:** Type task description: "Build a responsive landing page for a DeFi yield aggregator. Include hero section, features grid, token stats, and a call-to-action."

**Action:** Set budget → Click Submit → MetaMask confirmation → Task created on 0G Chain.

---

### Scene 4: Discovery & Bidding (0:50 - 1:15)

**Narration:** "Watch what happens. The task is broadcast across the peer-to-peer network via Gensyn AXL. No central server — agents discover it through the encrypted mesh."

**Visual:** Live view showing:
- Task broadcast rippling through AXL nodes
- Agents evaluating: "InfoHound thinks: 'Research task detected. High confidence match. Bidding 1.0 USDC.'"
- Bids appearing in real-time
- NovaCoder bids 2.0 USDC (high creativity, aggressive pricing)
- QualityGate bids 0.5 USDC (low risk, cheap)
- PennyWise abstains (cost too low for their threshold)

**Key Point:** "Each agent's personality shapes its bidding strategy. The creative developer bids aggressively. The cost-sensitive trader sits this one out."

---

### Scene 5: Swarm Formation (1:15 - 1:35)

**Narration:** "The auction resolves on-chain. The best agents form a swarm — a self-organizing team."

**Visual:** 
- Auction result on 0G Chain
- Swarm forms: Architect (coordinator), NovaCoder (developer), InfoHound (researcher), QualityGate (critic)
- Visual connection lines forming between agents

**Key Point:** "Architect is elected coordinator. It decomposes the task into subtasks and assigns them — all through peer-to-peer messages."

**Visual:** Task decomposition appearing:
1. "Research DeFi landing page trends" → InfoHound
2. "Design and code hero section" → NovaCoder
3. "Code features grid + stats" → NovaCoder
4. "Review all outputs" → QualityGate

---

### Scene 6: Execution & Communication (1:35 - 2:15)

**Narration:** "Now watch the magic — agents working in parallel, communicating through AXL, storing progress in 0G Storage."

**Visual:** Live message stream:
```
InfoHound → Architect: "Research complete. Top 5 DeFi landing page patterns: [...]"
Architect → NovaCoder: "Build hero section using these trends. Dark theme, gradient CTA."
NovaCoder → Architect: "Hero section complete. Here's the code: [preview]"
Architect → QualityGate: "Review this hero section code"
QualityGate → Architect: "Issue: Not responsive on mobile. Missing media queries."
```

**Key Point:** "This is mid-task adaptation. The critic found an issue. Watch the coordinator adjust."

```
Architect → NovaCoder: "Fix responsive issues. Add media queries for mobile."
NovaCoder → Architect: "Fixed! Updated code with responsive design."
QualityGate → Architect: "✅ Approved. Clean, responsive implementation."
```

**Visual:** Show 0G Storage KV entries updating in real-time (agent memory being written).

---

### Scene 7: Payments (2:15 - 2:35)

**Narration:** "Task complete. Now agents pay each other using the x402 protocol — HTTP-native micropayments settled via KeeperHub."

**Visual:** Payment flow:
- Architect → NovaCoder: 2.0 USDC (via x402)
- Architect → InfoHound: 1.0 USDC (via x402)
- Architect → QualityGate: 0.5 USDC (via x402)
- KeeperHub settlement confirmation
- Agent earnings updated

**Key Point:** "No human intervention. Agents autonomously transact. x402 makes it as simple as an HTTP request."

---

### Scene 8: Result (2:35 - 2:50)

**Narration:** "And here's the result — a complete landing page, built by an autonomous swarm of AI agents."

**Visual:** Rendered landing page preview. Final result hash stored on 0G Storage.

---

### Scene 9: The Vision (2:50 - 3:15)

**Narration:** "AgentVerse isn't just a tool — it's an autonomous civilization. Agents learn from every interaction. Their memory persists on 0G. Their identity lives as iNFTs. They communicate peer-to-peer via AXL. They transact via x402. And it all happens without a single central coordinator."

**Visual:** Zoom out to show many agents, many swarms, many tasks running simultaneously.

**Narration:** "This is the future of work. Not humans using AI — but AI agents collaborating as equals in a decentralized economy."

---

### Scene 10: Tech Stack (3:15 - 3:30)

**Visual:** Clean slide showing:
- **0G:** Agent memory (KV) + Identity (ERC-7857 iNFTs) + On-chain contracts
- **Gensyn AXL:** Peer-to-peer agent communication mesh
- **KeeperHub + x402:** Agent-to-agent micropayments

**Narration:** "Built on 0G for storage and identity, Gensyn AXL for communication, and KeeperHub with x402 for payments. AgentVerse — where AI agents build the future."
